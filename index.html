Overview

Over the course of this 1-day training class you will learn to run water use scenarios with the Wood River Valley (WRV) groundwater-flow model.

Files

The groundwater-flow model is described in the associated model report and model archive.

The processing programs and unprocessed data sets for the groundwater-flow model are bundled together in an R package named wrv. The file repository for this package is located at https://github.com/USGS-R/wrv. Included in the wrv package is MODFLOW-USG version 1.2, a U.S. Geological Survey (USGS) computer code that solves the groundwater-flow equation.

Files used in this training class are available on a wrv-training repository. There will also be a USB-thumb drive distributed in class that contains the repository file structure, and includes many other files that result from intermediate steps in data processing. These additional files will be helpful if your computer is unable to execute some of processing steps because of memory limitations or time constraints.

Software

Installation instructions for most of the software used in this class is located at https://jfisher-usgs.github.io/r-workshop/software.html.

A beta version of the R-package wrv also needs to be installed from its binary distribution file, located on the USB drive (“wrv_1.0.3.zip”). This version of the package will soon be made available from the Geological Survey R Archive Network (GRAN). The new package version does not change model output or analysis results. One major change is the removal of general-use functions from wrv. These removed functions were bundled together in a new R package called inlmisc, available on the Comprehensive R Archive Network (CRAN).

First remove any previous version of the wrv package you may have installed by opening an R session and typing the following command:

remove.packages("wrv")

To install the inlmisc package and its dependencies from CRAN use

install.packages("inlmisc", repos = "https://cloud.r-project.org/", dependencies = TRUE)

And to install the beta version of the wrv package use

install.packages(file.choose(), repos = NULL)

You will be prompted to select the location of the local ZIP file “wrv_1.0.3.zip”.
